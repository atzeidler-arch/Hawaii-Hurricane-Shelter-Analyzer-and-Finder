# Hawaii Hurricane Shelter Data Analyzer

## Author
Andrew Zeidler

## Description
This project reads a CSV dataset of hurricane shelters in Hawaii and performs basic data analysis and user-driven queries. The script processes structured data, summarizes key shelter features, and allows users to search for shelters by zip code.

## Features
- Counts total number of shelters
- Summarizes shelter totals by island
- Counts total special-needs shelters
- Counts total pet-friendly shelters
- Allows users to search shelters by zip code
- Displays all shelter names and locations within the inputted zip code

## Purpose
This project was developed to demonstrate foundational Python skills in:
- CSV data ingestion
- Data cleaning and preprocessing
- Dictionary-based data storage
- Iteration and conditional logic
- Basic data analysis and filtering
- User input handling

## Files
- `shelter_analyzer.py` — main Python script
- `hawaii_shelters.csv` — dataset of hurricane shelters

## How to Run
1. Ensure both files are in the same directory
2. Open a terminal in that directory
3. Run the script:

```bash
python shelter_analyzer.py
